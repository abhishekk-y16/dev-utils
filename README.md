# Developer Utilities Hub

A lightweight, frontend-only collection of small developer utilities built with HTML, CSS and JavaScript.

## Project overview

This repository contains multiple small tools useful for daily development tasks. Each tool is implemented as a small JS module and presented in a clean card-based UI.

Tools included:

- Password Generator
- Word Counter
- JSON Formatter
- Base64 Encoder / Decoder
- Color Generator
- Regex Tester
- Timestamp Converter
- URL Encoder / Decoder

Tech stack

- HTML, CSS, Vanilla JavaScript (ES modules)
